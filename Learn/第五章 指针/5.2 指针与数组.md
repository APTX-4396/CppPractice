# 5.2.1 指向数组的指针变量
定义的一般格式为：  
数据类型 * 指针变量 = 数组名；  
或者
数据类型 指针变量 = &数组名[首地址];  
**数组的首地址也称为数组的指针，指向数组的指针变量称为数组指针变量，可以通过改变数组指针的值访问数组的每个元素**
